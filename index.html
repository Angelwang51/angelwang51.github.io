Fire Blog

A blog based on AngularJS and Firebase

A blog based on angular.js and firebase. The theme of the blog is a pure HTML&CSS responsive grid style. User can also customize their own blog style. I used Markdown as an text editor. The blog will generate the preview at the same time when the user edits markdown. Fireblog is SPA (single page application) built with AngularJS and Foundation responsive font-end framework.
